# dotfiles

My personal dotfiles, managed with [chezmoi](https://www.chezmoi.io/).

This repo is meant to be a home for the config I actually use every day. Feel free to look around and copy anything useful.

## Philosophy

These are a few things I try to optimize for:

- clarity over cleverness
- consistency over novelty
- practical defaults over excessive abstraction
- enough documentation to be useful later, but not so much that it becomes a chore to maintain

Some files are intentionally minimal. Others are a little more detailed when that helps future maintenance.

## Notes

This is *not* intended to be a universal template. It's just a working setup that I want to keep pleasant to use for years.

If something looks oddly specific, it's probably because it solved a real problem and made the repo easier to live with.
